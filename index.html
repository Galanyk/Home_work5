Селекторы

Селекторы элемента

Универсальный селектор

Универсальный селектор обозначается * (звездочкой) и предназначен для выбора всех элементов.

Селекторы по названию тега

Предназначен для выбора элемента по имени тега. Пример: cелектор р выберет все элементы р на странице

Селекторы по классу

Предназначен для выбора элемента по классу (значению атрибута class). Пример: селектор .center выберет все элементы, имеющие класс center.

Селекторы по идентификатору (id)

Предназначен для выбора элемента по идентификатору (значению атрибута id). Синтаксис - #имяИдентификатора. Пример: для элемента, имеюшего в качестве значения атрибута id значение footer. Селектор #footer выберет элемент, имеющий тдентификатор footer.

Селекторы по нескольким классам

Cелектор применяется к элементу, содержащему несколько CSS классов. Пример: селектор:.first-class-name.second-class-name

Селекторы по вложенности

Возможность вкладывать селекторы в другие селекторы
